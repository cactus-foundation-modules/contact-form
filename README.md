# Cactus Contact Form Module

A configurable contact form module for [Cactus](https://github.com/usersaynoso/cactus-foundation).

Provides a Puck page builder block, admin submission inbox, markdown reply composer,
notification emails, auto-reply, Turnstile spam protection, rate limiting, GDPR consent,
and CSV export.

## Installation

Add as a git submodule to your Cactus installation:

    git submodule add https://github.com/usersaynoso/cactus-module-contact-form modules/contact-form

Then activate the module from the Cactus admin panel under Modules.

## Configuration

Once activated, configure the module under Admin > Contact > Settings.
